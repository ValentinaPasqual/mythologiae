MythoLOD dataset

Intoduction

Mythologiae source data has been originally extracted from Mythologiae Collection. Such digital collection has been designed and implemented by FRAMELab (University of Bologna) as a collaborative project between researchers and students. The digital collection today "raccoglie" (but still it is an ongoing project) 4545 cultural object with different characteristics (e.g types, periods, authors, collocations). The collection pivotal element is given by the fact that each item represents scenes from mythological and/or classical themes. Moreover, the most peculiar characteristic of such collection is that firstly, to each cultural object is associated a conceptual category (e.g. macrothemes describing the abovementioned scenes) and secondly, it is associated to a literary source. Each cultural object then, shares the meaning of its depicted scene with one or more themes narrated in a literary source.

This section presents briefly the process of conversion of data from database format to LOD and then presents an example of how a typical assertion has been represented with MIMA datamodel.

Data conversion and Approach

Mythologiae data from FRAMELab has stored into a database, which is the source of the browsable collection. We took the database data as the input of the conversion process in LOD format.
DB data has been automatically (and manually when needed) cleaned from typos and inaccurances, disambiguised, reconciled against external LOD sources (e.g. Wikidata, VIAF

) and finally converted in RDF format (particularly using namedgraphs - quads, for more information about how the datamodel has been defined please see datamodel section).

Case study: How a prototypical cultural object is modelled in the dataset across the four layers (for more information about the four layers please see the datamodel section)

The case study in natural language:

  • Asserzione449 graph: L’oggetto museale 449 rappresenta la scena di medea figlicida. La scena fonte della rappresentazione viene individuata in “Seneca, Medea, vv.893-1027” (cit/22). La scena viene poi ripresa in altre opere seguenti: “Lenormand Henri, Rene Asie” e “Dolce Lodovico, Medea”. All’oggetto 449 sono associate delle note, delle keywords, l’immagine digitale che lo ritrae e la descrizione della sua rappresentazione (valore concettuale dell’opera);
  • FactualData graph: L’oggetto museale 449, “Fuga di Medea”, è un esempio di pittura vascolare prodotto nel 400 a.C (arte greca - età classica). Oggi è custodito nel Museum of Art di Cleveland (USA);
  • Provenance449 graph: L’asserzione449 è generata da un atto interpretativo di tipo ‘associazione di fonti’ e con criterio (approccio) iconografico. L’atto interpretativo è stato attuato da Hubert Gamba in data 11/04/2019;
  • PublicationInfo449 graph: L’intera nanopublication449 (assertion + provenance + publicationInfo) è stata pubblicata in data 24/08/2020 a cura del laboratorio di ricerca /DH.arc.
